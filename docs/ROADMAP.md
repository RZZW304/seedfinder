# Roadmap

## Current Version: v0.1.0

### Completed Features
- Professional PyQt5 GUI with dark theme
- Core seedfinding logic with SeedFinder and FastSeedFinder classes
- Multi-threaded search with progress tracking
- Configuration options for search parameters
- Results export to JSON
- Fast mode for 48-bit search
- Build scripts for Windows and Linux
- Comprehensive documentation
- Unit tests
